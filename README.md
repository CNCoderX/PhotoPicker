# PhotoPicker

To get a Git project into your build:

Step 1.Add it in your root build.gradle at the end of repositories:
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
Step 2. Add the dependency
	dependencies {
	        compile 'com.github.CNCoderX:PhotoPicker:1.0.0'
	}
